# 六爻排盘解卦 Divination

基于传统**京房纳甲六爻**理论的排盘与解卦 PWA 工具。Vue 3 + Vite 5 + Pinia，原生 Canvas 渲染，anime.js 动画。

## 功能

- 🎲 自动连续摇卦（龟壳铜钱动画）
- 📊 完整装卦（纳甲/六亲/六神/世应/旬空/伏神）
- 🔍 智能解卦（13 类问题识别，双版本断语，零术语建议）
- 📤 PNG/PDF 导出
- 📱 PWA 离线支持

## 技术栈

Vue 3.4 · Vite 5.4 · Pinia 2.1 · Vue Router 4.3 · anime.js 4.5 · Canvas · PWA

## 快速开始

```bash
npm install
npm run dev      # 开发服务器 → http://localhost:5173
npm run build    # 生产构建 → dist/
npm run test     # Vitest 全量测试 (63 tests)
```
